"Display: inline-block;" It will behave like a inline element but in addition we can apply custom weight and height.
Inline elements are displayed on the same line(Span).
Inline elements (Span) will not take width and height properties. It will use content width. And we can't use custom margin in case of inline elements.


Difference Between Block and Inline Elements

Block Elements Inline Elements
1. Try to render in new line. Try to render in same line.
2. By Default it will take width extreme left and extreme right. It's width depends on content width. It will take the content width.
3. Here we can apply custom width and height. Here we can't apply custom width and height.
4. Here we can apply margin and padding. Here we can also apply margin and padding but only horizontally not vertically.
Note: You can change the display property of any element using CSS to make it behave like a block or inline element (We can change block elements into inline elements and vice versa).
Like we can change span to block using "display:block".
